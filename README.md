# **LG G5 Louder**
## Description
Makes the LG G5 (h850) headphone output louder when used with LineageOS 14.1 (Android 7.1.2)
## Changelog
- *2018.09.01* - Second version, changed to Magisk API 17000
- *2018.04.08* - First version, works with LineageOS 14.1
